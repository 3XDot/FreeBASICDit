## 简要说明
这个编程语言是由FreeBASIC语言构建的

创造这个语言的初衷是为了使软件升级变得更加容易

语法类似于汇编语言

## 如何构建
Window x64环境下使用[VFB ide](http://www.yfvb.com/soft-48.htm)直接编译即可
## 示例代码
```
main:
VAR FF 
VAR G
LET FF=1
LET G=<CALL <print FF>>
FREE FF
RET  G
END main
```
